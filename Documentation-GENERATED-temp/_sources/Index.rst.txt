==================
LLMs.txt Generator
==================

:Extension key:
   llms_txt

   |today|

----

This TYPO3 extension generates ``llms.txt`` files for AI/LLM crawlers, providing
a compact index of your website with SEO metadata and instructions for accessing
page content in Markdown format. Optionally protect access with an API key.

..  note::

   This extension implements the `llmstxt.org specification <https://llmstxt.org/>`__.
